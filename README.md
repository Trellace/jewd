## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Once the  above command has run, then in another bash terminal in the same repo. Execute the following command.
```bash
node run server/server.js
```
